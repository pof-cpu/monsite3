Invite de commande windows

La commande dir : Lister (fichiers, dossiers).

  • dir /s affiche les sous-dossiers du dossier courant
  • dir /h affiche les dossiers cachés
  • dir mon_dossier>fichier.txt permet d'enregistrer la liste du répertoire mon_dossier dans le fichier texte fichier.txt
    • dans le fichier texte fichier.txt
  • dir mon_dossier\*.doc : permet d'afficher tous les fichiers .doc present dans le dossier mon-dossier
